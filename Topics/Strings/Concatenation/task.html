Concatenation

You have a []string slice words. It includes parts of a speech that somebody says.

Your job is to reconstruct the original speech by adding every new sentence to what you already have.

Take notice that this task uses the for loop statement to iterate over each word in the words string slice. However, don't get scared! to solve this task you don't need to know how the for loop works or what a slice is... Your only objective is to write the code to concatenate the speech and word variables!


Memory limit: 256 MB
Time limit: 8 seconds
